# Audio Assets

Tempatkan berkas audio final di folder ini dengan nama berikut:

- `gema-takbir-loop.mp3`: lantunan takbir utama yang dapat diputar berulang, disarankan di bawah 2 MB.
- `sfx-hover.mp3`: efek interaksi singkat untuk tombol dan kartu.

Nama file yang sama juga dicatat sebagai komentar di `index.html` dan `js/app.js`. Website tetap dapat dibuka tanpa MP3; panel pemutar akan memberi petunjuk ketika audio belum tersedia.
